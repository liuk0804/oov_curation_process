Introduction on OOV words

People have been posting on Social Media using language that combine spoken, written language. Text posted on social media they tend to be less grammatical, frequently contain spelling errors, unorthodox capitalization, and make frequent use of emoticons, abbreviations and hashtags. This research focused specifically on processing those non-standard English we refer as Out-of-dictionary words.

Research Highlight Analyzing OOV-words

  • Social Media Text is valuable data source for psycholinguistic analysis. Usage/Growth/personal discourse
  • Current Natural Language Processing Tools are not having issues processing unstructured text. Analyzing social media text requires specific consideration.
  • OOV words are often be ignored or over simplied in social media text analytics, only In-V words are captured and analyzed for trait analysis

We compare output of our proposed OOV-aware curation process with different text analytical approaches analyzing social media text, first using the most wide used pscyholinguistic dictionary Linguistic Inquiry and Word Count(LIWC), and one of most used python NLP library Natural Language Toolkit(NLTK). Result suggest following issues:

  1. dictionary based LIWC perform poorly on social media text with frequent OOV words usage
  2. current NLP tool having problem properly tokenizing unstructured social media text such as usage of url and emoticon
  3. behavior of intentionally capitalize or elongdate words are ignored
